# kahtaf.com

Kahtaf Alam's personal website, based on the [Astro Nano](https://github.com/markhorn-dev/astro-nano) theme.

## Tech Stack

Astro, Tailwind CSS, TypeScript, MDX

## Local Development

```bash
npm install
npm run dev       # starts dev server at localhost:4321
npm run build     # builds production site to ./dist/
```
